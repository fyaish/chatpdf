# ChatPDF

Ask anything to PDFs.
[Demo](https://chatpdfs.streamlit.app/)

## Installation

Developed using `python 3.12.2` on windows.

```bash
git clone https://github.com/fyaish/chatpdf/
```

## Create virtual env.

```bash
cd chatpdf
```

```bash
python -m venv venv
```


```bash
pip install -r requirements.txt
```

## Usage

```bash
streamlit run app.py
```
